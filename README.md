## üé® Customization

### Update Personal Information

1. **Header**: Edit `components/Header.tsx` to change the name
2. **Hero Section**: Update the hero content in `pages/index.tsx`
3. **About Section**: Modify the about text in `pages/index.tsx`
4. **Skills**: Update the skills array in `pages/index.tsx`
5. **Projects**: Modify the projects array in `pages/index.tsx`
6. **Contact**: Update email and social links in `pages/index.tsx` and `components/Footer.tsx`

### Update Resume

Replace the file `public/Dineshresume.pdf` with your own resume PDF.

### Update Social Links

Edit the social links in:
- `components/Footer.tsx` - Footer social icons
- `pages/index.tsx` - Contact section social icons

### Styling

- Global styles: `styles/globals.css`
- Tailwind config: `tailwind.config.js`
- Component-specific styles use Tailwind classes
